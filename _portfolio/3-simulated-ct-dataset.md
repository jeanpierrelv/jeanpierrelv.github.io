## Simulated CT Dataset Generator for Eggs

Framework for generating synthetic 3D CT images of chicken eggs that accurately simulate real tomographic data with different density variations, enabling robust deep learning model training.

**Technologies:** Python, DICOM, 3D Image Processing, CT Simulation

**Key Features:**
- Realistic DICOM format output
- Variable egg size and internal structure variations
- Labeled masks for segmentation training
- 99.4% accuracy with U-Net and FCN models

**Related Publication:**
- Generation of Simulated Dataset of Computed Tomography Images of Eggs and Extraction of Measurements Using Deep Learning (ESANN Conference, 2024)
